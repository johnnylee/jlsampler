JLSampler: Installation Guide

Installing from source

JLSampler is written in Go. Installing the Go compiler is beyond the scope of this document. See the previous link for details.

JLSampler depends on flac, JACK, and ALSA. In order to build from source you'll need the development packages available for your system.

With the prerequisets installed, use the go get command to fetch the source. 

go get github.com/johnnylee/jlsampler

Inside the directory johnnylee/jlsampler/jlsampler run go build to build the binary.

Installing a binary

The distributed binaries should run on your system if you have the appropriate libraries installed: flac, JACK, and ALSA.

See the documentation page for further details.
