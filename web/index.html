JLSampler

JLSampler aims to be a simple and robust open-source sampler for Linux with a primary focus on playability using a midi keyboard.

2014-05-17: Sonatina Symphonic Orchestra

Samples from the Sonatina Symphonic Orchestra have been added to the samples page. These single-layer samples take advantage of features in the 0.91 release.

2014-05-16: 0.91 Released

Version 0.91 has been released. See the download page for the binary. See changes here.

2014-05-13: Added Maestro Concert Grand Samples

Thank you to Mats Helgesson for permission to repackage his wonderful piano samples. See the samples page for the download link.

2014-05-11: Initial Release

The first public version of JLSampler (0.9) is available. Being pre-1.0, it is possible that incompatible changes will be made to the sample format, the configuration file format, or the interface.

Please feel free to try out this release and report any issues you find.
